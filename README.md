## Running tests:
- Activate virtual environment:
	```ssh
	$ virtualenv venv # Create virtual environment if does not exist
	$ source venv/bin/activate
	```

- Install dependencies
	```ssh
	$ pip install -r requirements.txt
	```

- Run tests
	```ssh
	$ pytest
	```
